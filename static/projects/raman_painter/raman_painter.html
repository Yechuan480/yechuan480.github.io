Raman Painter

Program Overview

Description: Raman Painter is a Python script for visualizing Raman spectroscopy data, converting raw spectra into colorful plots for analysis.

Functionality:

  • Reads Raman spectra data from text or CSV files.
  • Generates customizable plots with Matplotlib.
  • Highlights specific peaks or regions (planned feature).

Data Format Requirements:

  • Input: CSV or text file with two columns (wavenumber, intensity).
  • Example: "wavenumber,intensity\n400,0.1\n450,0.3"
  • Output: PNG or interactive plot.

Sample Code


import pandas as pd
import matplotlib.pyplot as plt

def plot_raman_spectra(file_path):
    data = pd.read_csv(file_path)
    plt.plot(data['wavenumber'], data['intensity'], color='blue')
    plt.xlabel('Wavenumber (cm⁻¹)')
    plt.ylabel('Intensity')
    plt.title('Raman Spectrum')
    plt.grid(True)
    plt.show()

# Example usage
file_path = "raman_data.csv"
plot_raman_spectra(file_path)
