Documents Hunter

Program Overview

Description: Documents Hunter is a Python tool designed to search and organize files on your local system based on specific criteria (e.g., file type, keyword, date).

Functionality:

  • Scans directories recursively for documents.
  • Filters files by extension (e.g., .pdf, .docx).
  • Extracts metadata or content keywords (planned feature).

Data Format Requirements:

  • Input: Directory path (string).
  • Optional: List of file extensions (e.g., ['.pdf', '.txt']).
  • Output: List of file paths or JSON report.

Sample Code


import os

def find_documents(directory, extensions=None):
    if extensions is None:
        extensions = ['.pdf', '.docx', '.txt']
    found_files = []
    for root, _, files in os.walk(directory):
        for file in files:
            if any(file.endswith(ext) for ext in extensions):
                found_files.append(os.path.join(root, file))
    return found_files

# Example usage
directory = "C:/Users/YiFan/Documents"
files = find_documents(directory)
for file in files:
    print(file)
