Documents Hunter

Program Overview

Description: Documents Hunter is a Python tool designed to crawl and download files (e.g., PDF, ZIP) from a specified website, organizing them by file type into local directories.

Functionality:

  • Extracts download links from webpages, supporting multiple file formats (PDF, ZIP, DOC, etc.).
  • Automatically creates subfolders categorized by file type.
  • Downloads files and saves them to a specified directory with cleaned filenames.

Data Format Requirements:

  • Input: Target website URL (string, e.g., "https://example.com").
  • Output: Files saved by type in local directories (e.g., "path\to\downloads\pdf\").
  • Dependencies: Requires Python libraries: requests, beautifulsoup4.

Code and Execution

Code:

import requests
from bs4 import BeautifulSoup
import os
import urllib.parse
import re
def download_file(url, save_path):
    """Download a file and save it to the specified path"""
    try:
        response = requests.get(url, stream=True)
        response.raise_for_status()
        with open(save_path, 'wb') as f:
            for chunk in response.iter_content(chunk_size=8192):
                if chunk:
                    f.write(chunk)
        print(f"Successfully downloaded: {save_path}")
    except requests.RequestException as e:
        print(f"Download failed {url}: {e}")
def crawl_downloads(url, base_save_dir=r"C:\Users\Example\Downloads"):
    """Crawl download links from a website and save by file type"""
    if not os.path.exists(base_save_dir):
        os.makedirs(base_save_dir)
    try:
        headers = {'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/91.0.4472.124 Safari/537.36'}
        response = requests.get(url, headers=headers)
        response.raise_for_status()
        soup = BeautifulSoup(response.text, 'html.parser')
        download_links = {}
        for link in soup.find_all('a', href=True):
            href = link['href']
            link_text = link.text.strip()
            file_ext = None
            match = re.search(r'\.(\w+)$', href)
            if match:
                file_ext = match.group(1).lower()
            if file_ext and file_ext in ['pdf', 'zip', 'doc', 'docx', 'jpg', 'png', 'xls', 'xlsx', 'txt']:
                file_name = link_text if link_text else os.path.basename(href.split('?')[0])
                file_name = re.sub(r'[<>:"/\\|?*]', '', file_name)
                if not file_name:
                    file_name = f"unnamed.{file_ext}"
                elif not file_name.endswith(f".{file_ext}"):
                    file_name += f".{file_ext}"
                download_links[href] = (file_name, file_ext)
        for href, (file_name, file_ext) in download_links.items():
            full_url = urllib.parse.urljoin(url, href)
            sub_dir = os.path.join(base_save_dir, file_ext)
            if not os.path.exists(sub_dir):
                os.makedirs(sub_dir)
            save_path = os.path.join(sub_dir, file_name)
            print(f"Found download link: {full_url} -> {save_path}")
            download_file(full_url, save_path)
    except requests.RequestException as e:
        print(f"Failed to access website {url}: {e}")
    except Exception as e:
        print(f"An error occurred: {e}")
if __name__ == "__main__":
    target_url = "https://example.org/resources"
    crawl_downloads(target_url)

Note: Replace highlighted values (yellow) with your own URL and save path.

Execution:

1. Install required libraries:

pip install requests beautifulsoup4

2. Save the code as documents_hunter.py, then run in the command line:

python documents_hunter.py

Modify target_url and base_save_dir as needed before running.

Example

Example: Using NASA Lunar Sample Compendium as the target website.

Steps:

  • Set target_url to "https://curator.jsc.nasa.gov/lunar/lsc/".
  • Run python documents_hunter.py.
